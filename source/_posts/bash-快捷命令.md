## 历史记录相关快捷命令
参见：`man history`。下面节选一部分：

| 命令    | 说明                            |
| ------- | ------------------------------- |
| `!!`    | 执行上一条命令                  |
| `!foo`  | 执行以 `foo` 开头的命令         |
| `!?foo` | 执行包含 `foo` 的命令           |
| `!n`    | 执行第 n 条命令                 |
| `!-n`   | 执行倒数第 n 条命令             |
| `!$`    | 上一条命令最后一个参数          |
| `!^`    | 上一条命令第一个参数            |
| `!:n`   | 上一条命令第 n 个参数           |
| `!:x-y` | 上一条命令从 x 到 y 的参数      |
| `!:n*`  | 上一条命令从 n 开始到最后的参数 |
| `!*`    | 上一条命令的所有参数            |
